## **Jogo do número secreto**

Esse projeto foi criado para jogarmos via web o jogo do Número secreto.

Nele você receber a instrução para chutar um número dentro de um range.

Você digitará seu número e o programa irá dizer se esse chute estará abaixo do número secreto ou acima.

Após acertar, e iniciar um novo jogo, o antigo número secreto deixará de ser selecionado.
